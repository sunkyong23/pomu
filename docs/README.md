## Upload to GitHub Pages
1. Open the `pomu` GitHub repository.
2. Create a `docs` folder in the repository root.
3. Upload every file and folder from this ZIP into `docs`.
   - Final path should be `docs/index.html`, `docs/privacy.html`, etc.
4. Open **Settings → Pages**.
5. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
6. Save and wait for deployment.

Expected URLs:
- Home: `https://sunkyong23.github.io/pomu/`
- Privacy Policy: `https://sunkyong23.github.io/pomu/privacy.html`
- Support: `https://sunkyong23.github.io/pomu/support.html`
